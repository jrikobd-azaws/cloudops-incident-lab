# Handover Template

## Purpose

This template is used to support clear shift-to-shift or team-to-team continuity during an active or recently mitigated incident.

The aim is to provide enough context for the next owner to continue safely without repeating checks or losing track of current risk.

## Handover quality guidance

A good handover should make the following clear:

- what is happening now
- what has already been checked
- what is still uncertain
- what risk remains open
- what the next owner should do first

The handover should reduce duplicated effort and avoid vague status transfer.
